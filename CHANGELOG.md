## [1.0.0] - 2025-06-27

- Initial release.

### Added

- A `matlab-run-cell.runCell` command to run the current cell.
- A `matlab-run-cell.focusTimeOut` configuration option to set the time-out for bringing the editor back into focus after running a cell.
